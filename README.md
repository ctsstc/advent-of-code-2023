# advent-of-code-2023

## Dependencies

To install dependencies:

```bash
bun install
```

## Running

To run:

```bash
bun run test
bun run test:watch
```

## Creating a New Day

```bash
bunx plop
```

## Bun Stuff

This project was created using `bun init` in bun v1.0.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
